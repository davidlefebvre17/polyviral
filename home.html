What is PolyViral?

PolyViral is an automated video publishing pipeline that transforms public Polymarket prediction market data into short-form vertical videos for TikTok. Every day, our system selects the most notable resolved markets, renders a 60-70 second video showing the winners, losers, and trade amounts, and publishes it directly to our own TikTok account.

The service is fully automated and does not collect, store, or process any personal data from TikTok users. It only posts content to the operator's own TikTok account.

How it works

  1. Fetch market data — We query the public Polymarket API for recently resolved prediction markets and the largest public trades.
  2. Select top stories — Our pipeline ranks the day's most notable results and selects the top 3 betting outcomes to feature.
  3. Render vertical video — Using Remotion, we render a 1080x1920 MP4 video (60-70 seconds) with market questions, outcomes, and trader results.
  4. Publish to TikTok — We upload the MP4 to TikTok via the Content Posting API (/v2/post/publish/video/init/ with FILE_UPLOAD) and publish with a caption and hashtags.

TikTok integration

Login Kit

Used once to authenticate the operator's own TikTok account so that videos can be posted programmatically. Scope: user.info.basic.

Content Posting API

Used to upload and directly post the rendered MP4 videos to the authorized account. Scopes: video.publish, video.upload.

Privacy-first

No TikTok user data is read, stored, or shared. The integration only writes content to the operator's own account.

Data sources

  • Polymarket public API — resolved market questions, outcomes, and public pseudonymous trade data
  • No data is collected from end users or TikTok viewers
  • No cookies, analytics, or tracking technologies are used on this site
